Projects

Explainable Image Aesthetic Factor Ranking

2025

Predicts 10-bin aesthetic score distributions from interpretable image features and ranks factor importance.

  • Built a lightweight MLP using sharpness, edges, exposure, saturation, and VGG-based statistics.
  • Ran ablation + perturbation analyses to quantify feature importance and produce concise visual summaries.

Real-Time Music-Driven Visual Modulation on Static Image

2025

A low-latency audio-to-visual system that modulates a static image in sync with music (volume / pitch).

  • Extracted RMS loudness and STFT-based pitch in real time.
  • Mapped audio features to zoom, hue, and brightness with calibrated curves + temporal smoothing.

Optimal Modulation of Interleukin-8 During Surgery Using Repertaxin

2025

Modeled and controlled intraoperative inflammation by optimizing Repertaxin dosing with state estimation.

  • Built a state-space model + feedback controller to modulate IL-8 levels during surgery.
  • Designed a Kalman filter estimator and personalized dosing policy; validated on 12-hour, 10-patient cardiac-surgery IL-8 time series in MATLAB/Simulink.

Accelerating the Design of Perovskite Solar Cells Using Active Learning

2023

Used machine learning + active learning ideas to accelerate perovskite material design through predictive modeling.

  • Applied models such as SVM and XGBoost with feature engineering, model selection, and hyperparameter tuning.
  • Focused on optimizing material design decisions via data-driven iteration.

Balls on an Elastic Band

2022

Investigated two ball-on-rubber-band phenomena (including equipotential-line behavior) via experimental design, modeling, and analysis.

  • Conducted literature review and translated core theoretical-physics concepts into an experimental plan.
  • Built mathematical models, analyzed results, and delivered informative presentations.
